# Sound Processing Lab 1

Лабораторная работа N1 по анализу и фильтрации аудиосигналов. Основной результат находится в Jupyter Notebook `notebooks/lab1_main.ipynb`; низкоуровневый код вынесен в пакет `src/signal_lab`.

## Структура

```text
datasets/                  # пользовательские аудиофайлы
notebooks/lab1_main.ipynb  # основной отчет и демонстрация
src/signal_lab/            # модули загрузки, анализа, фильтрации и графиков
THEORY.md                  # теория для защиты
IMPLEMENTATION_PLAN.md     # план реализации
AGENTS.md                  # требования к лабораторной
```

Директория `.old/` не используется. Обработанные аудио и графики автоматически не сохраняются: все результаты выводятся внутри ноутбука.

## Установка

Проект рассчитан на Python 3.11.

```bash
pyenv local 3.11.5
python -m venv .venv
source .venv/bin/activate
python -m pip install -e .
```

Если используется Poetry:

```bash
poetry install
poetry run python -m ipykernel install --user --name sound-processing-lab1
```

## Запуск ноутбука

```bash
jupyter lab notebooks/lab1_main.ipynb
```

В начале ноутбука находятся переменные путей:

- `SPEECH_PATH`
- `MUSIC_PATH`
- `SOUND_EFFECT_PATH`

Если в вашей копии проекта выбраны другие аудиофайлы, достаточно заменить эти переменные. Сэмплы должны быть уже подготовлены вручную; вырезание по таймкодам в коде не реализуется.

## Что демонстрирует ноутбук

Ноутбук содержит пять сценариев:

- розовый шум;
- речь с добавленным гулом 50 Гц;
- инструментальная музыка;
- сэмпл с резкими атаками;
- влияние порядка обработки при наличии вейвлет-денойзинга.

В каждом сценарии показаны audio players, осциллограммы, спектры или PSD, STFT-спектрограммы, RMS и дополнительные признаки, релевантные конкретному сигналу.
